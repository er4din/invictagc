== Description ==

Janey Press is a stylish and modern WordPress theme, tailored for creating personal, travel, beauty, lifestyle, news, or photography blogs and websites. It supports seamless customization via the WordPress Live Customizer, offers WooCommerce compatibility, and ensures a mobile-friendly experience

Created by ThemeinProgress, https://www.themeinprogress.com
